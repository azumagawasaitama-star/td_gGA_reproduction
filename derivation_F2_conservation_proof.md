# 複素 TD-gGA の一貫規約と F2 厳密保存の証明（2026-07-07, Claude 導出）

対象: Guerci–Capone–Lanatà, PRR 5, L032023 (2023)（以下 PRR）。
規約の裏付け: Lanatà, Jülich Correl23 講義ノート第15章（`lanata.pdf`）Eqs. (77), (80)–(95), (141), (151), (156)–(161), (167)–(171)。

## 0. 結論（先に要点）

PRR の Eqs. (16)–(21) は **「転置されたホール表現」の規約**で書かれており、その規約で全式を
一貫させると **F2 = ⟨f f†⟩ − Δ は Λ=0 ゲージのまま厳密に保存される**（下の §3 で証明）。
これまでの実装（実対称版・複素版とも）は転置・共役が式ごとに混在しており、
その混在こそが「B=1 だけ完璧 / B=3 で F2・エネルギーがドリフト」の根本原因である
（B=1 では転置・共役の区別が消えるため）。

## 1. 論文の規約（転置の連鎖）

単一軌道・スピン因子化後、B 個の準粒子軌道。h(ω) ≡ ω RR† + Λ（Λ=0 ゲージでは ω RR†）。

- **n の定義（PRR Eq. 11）**: n_ab(ω) = ⟨η†_a η_b⟩ = [f(ωRR†+Λ)]_**ba**
  → n は「普通の Fermi 行列の転置」。初期条件も n₀ = [f(ωRR†₀+Λ_eq)]ᵀ とすること。
- **n の運動方程式（PRR Eq. 17 を添字展開）**: i ∂t n = ω [n, (RR†)ᵀ] = ω [n, (RR†)*]
  （標準規約 ñ=nᵀ で書けば i∂t ñ = [h, ñ] の普通の von Neumann 式。
   実装の従来形 dn/dt = −iω[n, RR†] は、n を「普通の Fermi 行列」とみなすなら
   **符号が逆**（+i[h,n] になっている）。B=1 と実対称力学では可換性・real 射影で不可視。）
- **Δ（PRR Eq. 21 / Jülich Eq. 94）**: Δ_ab = ⟨f̂_b f̂†_a⟩_Φ（bath **ホール**行列、消滅演算子が第2添字）
  かつ Δ = ∫ρ(ω) n(ω) dω。
- **F1 / R（PRR Eq. 20 / Jülich Eq. 95）**: ⟨c† f̂_a⟩_Φ = Σ_c R_c g_ca、g ≡ [Δ(1−Δ)]^{1/2}
  → ベクトル形で ⟨c†f⟩ = gᵀ R、**R = (gᵀ)⁻¹ ⟨c†f⟩ = (g⁻¹)ᵀ ⟨c†f⟩**（共役なし・転置あり）。
- **D（PRR Eq. 18）**: [R† ᵗn]_{1a} を展開すると (n R*)_a なので、B ≡ ∫ρω n dω として
  **g D = B R* ⇔ D = g⁻¹ B R\***（R に複素共役が付く！）。
- **H_emb の bath 項（PRR Eq. 4 / Jülich Eq. 82）**: Σ_ab Λc_ab f̂_**b** f̂†_**a**
  = Tr Λc − Σ_ab Λc_ab f̂†_a f̂_b → 正規順序での bath 一体行列は −Λc（転置なし）。
  添字順を f̂_a f̂†_b と書き間違えると bath 行列が −Λcᵀ になる（複素では別物）。
- **基底展開（PRR Eqs. 12–14 / Jülich Eqs. 86–88）**: Δ = Σ_s d_s **ᵗh_s**（転置基底）、
  Λ, Λc = Σ_s l_s h_s（非転置）。h_s は **B² 個の完全エルミート基底**、係数は実数。
- **Λc（PRR Eq. 19）**: l^c_s = −∂/∂d_s (Rᵀ g D + c.c.)。δΔ = ᵗh_s δd_s なので
  微分方向は **ᵗh_s**（h_s ではない！虚反対称基底では符号が反転する）。

## 2. Λc の閉形式（実装向け・基底ループ不要）

M ≡ D Rᵀ + (D Rᵀ)† = D Rᵀ + R* D†（エルミート、B×B）と置くと
Rᵀ g D + c.c. = Tr[g M]。Fréchet 微分の自己随伴性
（Loewner 核 φ_ij = (g_i−g_j)/(δ_i−δ_j) が対称）より

  l^c_s = −Tr[dg[ᵗh_s] M] = −Tr[ᵗh_s dg[M]]

基底の完全性から係数和が畳めて

  **Λc = −(dg[M])ᵀ**

ここで dg[M] は Δ の固有基底 {δ_i} で
  (dg[M])_ij = φ(δ_i, δ_j) M_ij,  φ(x,y) = (g(x)−g(y))/(x−y),  φ(x,x) = g'(x)
（g(x) = √(x(1−x))）。Δ を対角化 → M を変換 → 要素ごとに φ を掛ける → 戻す → 転置。
B² 基底のループも least_squares も不要で厳密。

## 3. F2 厳密保存の証明（Λ=0 ゲージのまま）

**QP 側**: i dΔ/dt = ∫ρω [n, (RR†)ᵀ] dω = [B, (RR†)ᵀ]。
(RR†)ᵀ = R* Rᵀ と g D = B R* を使うと

  i dΔ/dt = B R* Rᵀ − R* Rᵀ B = g D Rᵀ − R* D† g.

**埋め込み側**: Θ_ab ≡ ⟨f̂_b f̂†_a⟩_Φ。H_emb（上記の添字順）との交換子を計算すると
（U 項は f と可換、F1 と F2（Θ=Δ）を使用）

  i dΘ/dt = D Rᵀ g − g R* D† + [Λcᵀ, Δ].

**差**: i d(Θ−Δ)/dt = [Λcᵀ, Δ] − [g, D Rᵀ + R* D†] = [Λcᵀ, Δ] − [g, M]。
§2 の Λcᵀ = −dg[M] を代入すると、必要な恒等式は

  [Δ, dg[M]] = [g(Δ), M].

Δ の固有基底で両辺の (i,j) 成分:
  左辺 = (δ_i − δ_j)·φ(δ_i,δ_j)·M_ij = (g_i − g_j) M_ij = 右辺. ∎
（縮退 δ_i=δ_j でも両辺 0 で成立。）

よって **F2(t=0)=0 なら F2(t)=0 が厳密に成り立つ**。射影・Baumgarte 安定化・動的 Λ は
本来不要。Λ_eq≠0 の平衡初期条件から Λ=0 で発展させる論文のプロトコルは、
この式系なら拘束を破らない（「ゲージ混在」は問題ではなかった）。

## 4. 実装レシピ（状態 = Φ と n(ω) のみ）

各 RHS 評価で:
1. Δ = ∫ρ n dω（複素エルミートのまま！np.real 禁止）、Δ を対角化して g, φ 核を用意
2. R = (g⁻¹)ᵀ ⟨c†f⟩_Φ
3. B = ∫ρω n dω、D = g⁻¹ B R*
4. M = D Rᵀ + R* D†、Λc = −(dg[M])ᵀ（§2 の Loewner レシピ）
5. H_emb 構築: U/2(n̂−1)² + Σ_a (D_a c†f_a + h.c.) + Σ_ab Λc_ab f_b f†_a
6. Φ̇ = −i H_emb Φ、ṅ(ω) = −iω [n, (RR†)ᵀ]

初期条件: 全 gGA 平衡（Λ_eq ≠ 0）から n₀(ω) = [f(ωR₀R₀†+Λ_eq)]ᵀ、Φ₀ は
H_emb[D₀, Λc_eq]（Λc_eq = −Λ_eq − (dg[M₀])ᵀ、静的 Eq. 7）の基底状態。

## 5. 既存実装との差分（= バグの候補一覧）

| 式 | 一貫規約 | 既存実装 |
|---|---|---|
| n の EOM | −iω[n,(RR†)ᵀ] | −iω[n,RR†]（転置欠落） |
| D | g⁻¹ B R*（共役あり） | dR@(RᵀB) = g⁻¹Bᵀ R（共役なし） |
| R 再構成 | (g⁻¹)ᵀ⟨c†f⟩ | 要監査 |
| Λc 微分方向 | ᵗh_s（虚基底で符号反転） | h_s |
| Λc 基底 | 完全 B² エルミート | 実対称 B(B+1)/2（実版） |
| H_emb bath 添字順 | Λc_ab f_b f†_a | 要監査 |
| 初期 n₀ | 転置あり | 転置なし |
| real 射影 | しない | np.real 多数 |

sweep_conj.py の 36 通り総当たりで見つからなかったのは、Λc の微分方向・bath 添字順・
初期条件の転置が探索軸に入っていなかったため。正解は「全部を同じ規約で揃えた 1 点」。

## 6. コード監査との突き合わせ（2026-07-07 追記）— 特定されたバグ一覧

コードの n は「普通の Fermi 行列」n_code = f(ωRR†+Λ)、すなわち n_code,ab = ⟨η†_b η_a⟩
（論文の n の転置）。ffd_ij = ⟨f†_i f_j⟩、fdc_i = ⟨f†_i c⟩、op_bb[i,j] = f_j f†_i。
この規約に §1–4 を翻訳すると、正しい式系は：

- **Δ の引数はホール行列 Δ_h ≡ I − ffd**（g は補集合不変なので R, D には影響しないが
  Loewner 微分 dF は dF(I−Δ) = −dF(Δ) と符号反転する）
- **R = conj(g⁻¹ @ fdc)**（derivation_complex_TDVP.md §2 は正しかった）
- **D = g⁻¹ @ conj(B @ R)**（同 §3 は正しかった。§7 での「dR@(RᵀB) に訂正」が逆に誤り。
  実数の静的コードとの一致では共役の有無を判別できない）
  ※ H_emb 組み立て側の規約に注意：build_H_emb_cplx は D_p を f†c 側に掛けるので、
  そこに渡すのは conj(D_phys)（論文の D は c†f 側の係数）
- **Λc = −(dF_{Δ_h}[M])ᵀ = +(dF_{ffd}[M])ᵀ**、M = D Rᵀ + R* D†（完全エルミート基底 or
  Loewner 閉形式）。既存 TD コードとの差は「符号」と「転置」の両方
- **n の運動方程式: ∂t n = −iω[RR†, n] = +iω[n, RR†]**。
  既存の 'vn' 形 −iω[n,RR†] は**符号が逆**（実対称力学と B=1 では不可視）。
  'paper' 形 −iω[n,(RR†)ᵀ] は「論文の転置 n」用の式を n_code に適用したもので、これも誤り。
  **sweep の探索空間 {vn, paper} に正解 +iω[n,RR†] が入っていなかった。**
- **F2 診断: ‖(∫ρ n dω)ᵀ − (I − ffd)‖**（転置と補集合の両方が必要。従来の複素診断
  ffd − ∫ρn は虚部の符号が逆 → dF2/dt|₀=0.408 は診断側の転置誤りを含む可能性大）
- **初期 Λ_eq: ∫ρ f(ωR₀R₀†+Λ) dω = I − ffd(Φ₀) を解く**（従来は = ffd を解いており
  Λ_eq_npz ≈ −Λ_true と符号が反転。監査 E1(a) の観測と一致）
- E_qp の einsum は 'ab,ba'（行列積のトレース）。routeAplus の 'ab,ab' は複素で誤り。

### バグの可視性パターン（観測事実との整合）

| バグ | B=1 | 実対称 B=3 | 複素 B=3 |
|---|---|---|---|

「B=1 は全スキームで機械精度、B=3 だけ壊れる」「実対称版でも B=3 の F2/E がドリフト」

複素版はそれに加えて n-EOM 符号・D/R 共役・Λc 転置が重なっていた。

## 7. 最終版：電子規約での一貫式系（2026-07-07 確定・数値検証済み）

§6 の hole 規約への翻訳は1点だけ誤っていた。診断（`diag_conv.py`）で確定した事実：
**このコードベースの静的鞍点は電子規約 ∫ρf(ωRRᵀ+Λ) = ⟨f†f⟩ で閉じている**
（imp 系で残差 4.5e-6。埋め込み状態の構成に PH 変換が織り込まれているため、
論文の hole 規約がコードの演算子上では電子規約として現れる）。また
fix_gauge の変換 W = permmat @ phasemat @ transmatᵀ が imp→npz 系の写像であることを
機械精度（9e-16）で確認。

### 電子規約の一貫式系（§3 と同じ手法で導出、同じ Loewner 恒等式で閉じる）

コードオブジェクト（n = f(ωRR†+Λ) 標準 Fermi 行列、ffd_ij = ⟨f†_i f_j⟩、
fdc_i = ⟨f†_i c⟩、op_bb[i,j] = f_j f†_i、H_emb の hyb は D が f†c 側）で：

  Δe = ffd = I − ⟨op_bb⟩,   g = [Δe(1−Δe)]^{1/2}
  R  = g⁻¹ fdc                      （共役なし）
  B  = Σ_f w_f ω_f n_f              （複素のまま）
  D  = conj(g⁻¹ B R)                （f†c スロット係数）
  M  = R Dᵀ + D* R†
  Λc = +(dg_{Δe}[M])ᵀ               （符号プラス！ hole 版と逆 = dF の補集合反転で整合）
  ∂t Φ = −i H_emb Φ,   ∂t n = −iω[RR†, n]
  F2 = ‖ffd − Σ_f w_f n_f‖          （転置なし、厳密保存）

保存証明の核（§3 と同型）: K ≡ R Dᵀ とすると emb 側 hyb 項 = gK − K†g、
qp 側 = Y − Y†（Y = R⊗(BᵀR*)）で、D の上記選択により Kg = Y が厳密に成立。
残差 [g, K+K†] を Λcᵀ = dg[M] が恒等式 [Δ, dg[M]] = [g, M] で正確に打ち消す。∎

初期条件: Λ_eq = W Λ_imp Wᵀ（ゲージ回転、root-finding 不要）、n₀ = f(ωR₀R₀ᵀ+Λ_eq)。
F2_init ≈ 静的残差（~5e-6）で拘束面上に乗る。
